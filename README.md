# Utopixia CLI (`utx`)

`utx` is the command-line tool for Utopixia. It lets you initialize a project, track files, commit structural changes, push them to the network, and download them back.

## Install

For Linux amd64, install `utx` with:

```bash
curl https://raw.githubusercontent.com/renaud-b/utx/refs/heads/main/install.sh | bash
```

## Quick Start

```bash
utx init
utx login <wallet_path>
utx add <file>
utx commit "message"
utx push
```
